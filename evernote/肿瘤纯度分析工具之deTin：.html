肿瘤纯度分析工具之deTin：
DeTiN estimates tumor in normal (TiN)  based on tumor and matched normal sequencing data. 

1.软件的安装
        github地址：https://github.com/broadinstitute/deTiN
        或者pip安装： pip install deTiN

2.参数详解
         python deTiN.py 
        --mutation_data_path  mutation_statistics_file (MuTect call stats file (or similar variants file e.g. Strelka v.2.9.7 or greater)).
                e.g.        example_data/HCC_10_90.call_stats.pon_removed.txt 
        --cn_data_path    allelic copy number segmentation file (GATK4 AllelicCNV seg file).
                e.g.        example_data/HCC-1143_100_T-sim-final.acs.seg 
        --tumor_het_data    heterozygous SNP variant counts in the tumor sample. (GATK4 tumor het cov file).
               e.g.        example_data/HCC_10_90.tumor.hets.tsv 
        --normal_het_data    heterozygous SNP variant counts in the normal sample. (GATK4 normal het cov file).
                e.g.        example_data/HCC_10_90.normal.hets.tsv 
        --exac_data_path     pickle file of minor allele fraction > 0.01 ExAC sites.
                e.g.        example_data/exac.pickle_high_af 
        --output_name  sample name
                e.g.        10_percent_TiN_simulation 
        --indel_data_path mutation_statistics_file
                e.g.        example_data/MuTect2.call_stats.txt 
        --indel_data_type MuTect2 
        --output_dir      output directory
                e.g.        example_data/
