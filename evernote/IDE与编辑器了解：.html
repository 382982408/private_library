IDE与编辑器了解：
1）IDE是集成开发环境（Integrated Development Environment）缩写。
        IDE是一种个编程软件，是集成了程序员语言开发中会需要的一些基本工具、基本环境和其他辅助功能的应用软件。IDE一般包含三个主要组件：源代码编辑器（Editor）、编译器、解释器（Compiler、Interpreter）和调试器（Debugger）。

Sublime Text 和notepad 虽然默认都不支持FTP远程文件编辑，但是都是可以通过安装插件来实现的。
        许多编辑器不能直接打开远程ftp文件进行编辑，用起来非常麻烦，可以通过xftp、filezilla等ftp文件传输工具来进行工作，但是有一些编辑器能够通过插件的方式来进行远程文件编辑。

2）sublime text  安装sftp模块，进行远程文件编辑。
    2.1  control+shift+p 打开命令面板，install package  选中“ install package controller ”选项并回车。
    2.2  control+shift+p 打开命令面板，输入sftp 回车安装。
    2.3 setup server 设置sftp 参数。
    2.4 注意配置remote_path 目录，只会显示这个目录下的所有文件。
https://www.cnblogs.com/jialongfei/p/7640632.html

3）怎样用 Notepad++ 连接 FTP 进行文件编辑。
    3.1  菜单栏: 插件 --NppFTP-- 勾选“Show NppFTP Window”.
    3.2  设置:FTP 链接配置，如图中红色区域，填写FTP信息。
    3.3  保存之后，点击链接，链接成功之后就可以编辑文件了，点保存之后，会自动上传。
