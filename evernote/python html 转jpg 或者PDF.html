python html 转jpg 或者PDF

示例地址：
https://blog.csdn.net/xys430381_1/article/details/80519787
https://java.ctolib.com/imgkit.html

说明：此工作需依赖wkhtmltopdf、wkhtmltoimage工具，此工具下载地址（https://wkhtmltopdf.org/），将此工具安装并将工具地址写入环境变量；即可工作。

1）转pdf
import pdfkit
pdfkit.from_url('http://google.com', 'out.pdf')
pdfkit.from_file('test.html', 'out.pdf')
pdfkit.from_string('Hello!', 'out.pdf')
你也可以传递一个url或者文件名列表:
pdfkit.from_file(['file1.html', 'file2.html'], 'out.pdf')

2）转jpg
import imgkit
imgkit.from_url('http://google.com', 'out.jpg') 
imgkit.from_file('test.html', 'out.jpg') 
imgkit.from_string('Hello!', 'out.jpg')
